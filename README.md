# 微信对话生成器

## 项目说明
本仓库基于上游项目二次修改而来：  
上游仓库：https://github.com/zixiwangluo/wxdh

当前仓库地址：  
https://github.com/JX-Lab/WeChatSimulator

这是一个纯静态网页项目，核心入口是根目录 [index.html]，可以直接部署到 GitHub Pages、静态服务器或虚拟主机。

## 本仓库二改内容
- 保留原有文字、图片、语音、红包、转账等聊天记录生成能力
- 新增更接近微信聊天截图的 `语音通话` / `视频通话` 记录条块
- 支持设置通话状态：`已接通`、`未接通`、`已取消`
- 支持设置通话时长，并自动规范为 `mm:ss` 格式
- 补充 `.nojekyll`，更适合直接部署到 GitHub Pages
- 修正站点图标引用，便于在线预览

## 在线使用
如果你在 GitHub 仓库里开启 Pages：

1. 进入 `Settings`
2. 打开 `Pages`
3. 选择 `Deploy from a branch`
4. 分支选择 `main`
5. 目录选择 `/ (root)`

启用后，这个仓库通常会生成类似下面的地址：  
https://jx-lab.github.io/WeChatSimulator/

原版演示地址：  
https://zixiwangluo.github.io/wxdh/

## 本地使用
这是静态项目，不需要后端。你可以直接：

- 双击打开 `index.html`
- 或者放到任意静态服务器目录中访问
- 或者上传到 GitHub Pages / Nginx / Apache / 虚拟主机


## 功能介绍
微信对话生成器，是一款在线微信聊天对话制作的工具，它可以设置苹果或安卓状态栏，包括手机电量、手机时间等，还可以设置不同用户的角色，然后发送文字、语音、红包、转账等多种好玩的功能，可谓是一款娱乐制作神器。

## 更新说明
本仓库在原版基础上主要补充了更贴近真实聊天记录截图的通话消息能力，包括：

- 语音通话记录条块
- 视频通话记录条块
- 已接通 / 未接通 / 已取消 状态显示
- 通话时长显示
